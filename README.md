# Customer Retention & Churn Analysis

## Project Overview
This project analyzes Telco customer data to understand 
why customers leave and what keeps them engaged.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Dashboard)

## Dataset
- Source: Kaggle - Telco Customer Churn Dataset
- Records: 7,043 rows

## Key Insights
1. Overall churn rate is 26.5% - higher than industry average
2. Month-to-month contract customers churn the most (55%)
3. New customers (0-11 months) are most likely to churn (2,069)
4. Churned customers paid higher monthly charges (₹74 vs ₹61)

## Dashboard Features
- Overall Churn Rate (Pie Chart)
- Contract Type vs Churn (Pie Chart)
- Tenure vs Churn (Bar Chart)
- Monthly Charges vs Churn (Bar Chart)

## Business Recommendations
1. Offer discounts to convert monthly customers to yearly plans
2. Focus retention efforts on new customers in first year
3. Review pricing strategy for high-charge customers
